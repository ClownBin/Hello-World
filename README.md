# Hello-World

A sample Maven project for demonstrating CI/CD workflows with GitHub Actions.

## Features

- Maven build configuration
- GitHub Actions CI/CD workflow
- Automated package publishing

## Requirements

- Java 11 or higher
- Maven 3.6+

## Build

```bash
mvn clean package
```

## Publish

The project uses GitHub Actions to automatically publish packages to GitHub Packages when a release is created.
